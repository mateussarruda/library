# Library
Biblioteca - projeto do The Odin Project
# Funcionalidades
- Inserir livro

    Campos: titulo, autor, número de página e se o livros foi lido
    
    ![Campos com espaços para inserir título, autor, número de páginas e checkbox se o livro foi lido e o botão de adicionar](img/fields.png)
- Excluir livros

    Após inserir um livro é possivel apaga-lo clicando no botão de lixeira

    ![Imagem com as características do livro e com o botão de delete destacado](img/delete-button.png)
- Alterar status de leitura

    Se o livro estiver marcado como não lido pode-se clicar no checkbox para indicar que o livro já foi lido e vice-versa

    ![Imagem com as características do livro e com o checkbox de livro lido destacado](img/read-button.png)
